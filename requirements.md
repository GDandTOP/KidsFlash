# 🎮 플래시 키즈 - 유아용 모바일 게임 앱

## 1. 프로젝트 개요

- **앱 이름**: 플래시 키즈 (Flash Kids)
- **타겟 연령**: 1~7세 영유아 및 미취학 아동
- **플랫폼**: 모바일 웹앱 (React Native / Next.js 기반)
- **핵심 가치**: 안전하고, 직관적이며, 교육적 요소가 포함된 재미있는 게임

---

## 2. UI/UX 디자인 원칙

### 2.1 시각 디자인
- **색상**: 밝고 채도 높은 파스텔 + 원색 조합 (노랑, 하늘, 연두, 분홍, 주황)
- **폰트**: 둥글고 큰 글씨체 (최소 24px), 한글 지원
- **아이콘**: 둥근 모서리, 큰 터치 영역 (최소 64x64px), 귀여운 일러스트 스타일
- **배경**: 부드러운 그라데이션, 움직이는 구름/별/꽃 등 생동감 있는 요소
- **캐릭터**: 친근한 동물 마스코트 (곰, 토끼, 고양이 등)

### 2.2 인터랙션 디자인
- **터치 기반**: 탭, 드래그 앤 드롭 위주 (스와이프 최소화)
- **피드백**: 모든 터치에 즉각적인 시각/소리 피드백 (반짝임, 효과음)
- **보상 시스템**: 별, 스티커, 축하 애니메이션으로 성취감 부여
- **실패 없는 UX**: 오답 시 벌칙 없이 재시도 유도, 긍정적 피드백
- **자동 진행**: 일정 시간 무입력 시 힌트 표시

### 2.3 안전 설계
- **광고**: 현재 단계에서는 구축하지 않으나 추후 google ads를 통해 적용 진행 예정
- **타이머**: 선택적 플레이 시간 제한 기능
- **오프라인 지원**: 인터넷 없이도 플레이 가능

## 3. 화면 구성

### 3.1 홈 화면 (게임 선택)
```
┌─────────────────────────────┐
│        🌈 Kids Play         │
│     (로고 + 마스코트 캐릭터)    │
│                             │
│  ┌─────┐  ┌─────┐  ┌─────┐ │
│  │ 🎨  │  │ 🧩  │  │ 🎵  │ │
│  │색칠  │  │퍼즐  │  │음악  │ │
│  │놀이  │  │맞추기│  │놀이  │ │
│  └─────┘  └─────┘  └─────┘ │
│                             │
│     ┌─────┐  ┌─────┐       │
│     │ 🎈  │  │ 🔢  │       │
│     │풍선  │  │숫자  │       │
│     │터트리기│ │세기  │       │
│     └─────┘  └─────┘       │
│                             │
│  ⚙️ 설정                     │
└─────────────────────────────┘
```

- 게임 카드는 크고 둥근 사각형, 귀여운 아이콘 + 큰 텍스트
- 카드 선택 시 바운스 애니메이션
- 배경에 느릿느릿 움직이는 구름, 별 파티클

### 3.2 게임 공통 UI
- **상단 바**: 홈 버튼 (좌측), 현재 게임 아이콘 (중앙), 별 카운터 (우측)
- **하단**: 힌트 버튼, 다음/이전 레벨 화살표
- **클리어 화면**: 별 3개 + 축하 애니메이션 + "다음" / "홈으로" 버튼

## 4. 게임 상세 (5종)

### 🎨 게임 1: 색칠 놀이 (Color Play)
| 항목 | 내용 |
|------|------|
| **설명** | 빈 도안을 손가락으로 터치하여 색칠하는 게임 |
| **조작** | 하단 팔레트에서 색상 선택 → 영역 탭하여 색칠 |
| **도안** | 동물, 과일, 탈것 등 12종 이상 |
| **난이도** | 영역 수 3개(쉬움) ~ 10개(어려움) |
| **보상** | 완성 시 그림이 살아 움직이는 애니메이션 |
| **교육 요소** | 색상 인지, 미세 근육 발달 |

### 🧩 게임 2: 퍼즐 맞추기 (Puzzle Match)
| 항목 | 내용 |
|------|------|
| **설명** | 조각난 그림을 드래그하여 올바른 위치에 맞추는 게임 |
| **조작** | 퍼즐 조각을 드래그 앤 드롭 |
| **종류** | 동물, 음식, 자동차 등 그림 퍼즐 |
| **난이도** | 2x2(쉬움) → 3x3(보통) → 4x4(어려움) |
| **보상** | 퍼즐 완성 시 그림 관련 동물 소리/효과 재생 |
| **교육 요소** | 공간 인지, 문제 해결력, 집중력 |

### 🎵 게임 3: 음악 놀이 (Music Play)
| 항목 | 내용 |
|------|------|
| **설명** | 화면의 악기를 터치하여 소리를 내고 음악을 만드는 게임 |
| **모드 1** | 자유 연주 - 피아노/실로폰/드럼 자유롭게 연주 |
| **모드 2** | 따라하기 - 빛나는 순서대로 터치하여 멜로디 완성 |
| **악기** | 피아노(8건반), 실로폰, 드럼, 탬버린 |
| **보상** | 멜로디 완성 시 동물 캐릭터들이 춤추는 애니메이션 |
| **교육 요소** | 리듬 감각, 청각 발달, 순서 기억력 |

### 🎈 게임 4: 풍선 터트리기 (Balloon Pop)
| 항목 | 내용 |
|------|------|
| **설명** | 올라오는 풍선을 터치하여 터트리는 게임 |
| **조작** | 올라오는 풍선을 탭하여 터트리기 |
| **모드 1** | 자유 모드 - 풍선을 마음껏 터트리기 |
| **모드 2** | 미션 모드 - 지정된 색상/모양 풍선만 터트리기 |
| **풍선 종류** | 빨강, 파랑, 노랑, 초록 + 별/하트 특수 풍선 |
| **보상** | 터트릴 때 반짝이 파티클 + 재미있는 효과음 |
| **교육 요소** | 색상 구분, 반응 속도, 선택적 주의력 |

### 🔢 게임 5: 숫자 세기 (Number Count)
| 항목 | 내용 |
|------|------|
| **설명** | 화면에 등장하는 사물을 세고 올바른 숫자를 선택하는 게임 |
| **조작** | 화면의 사물 개수를 세고 → 하단 숫자 버튼 탭 |
| **문제 유형** | "사과가 몇 개일까요?" → 사과 3개 그림 → 숫자 선택 |
| **난이도** | 1~3(쉬움) → 1~5(보통) → 1~10(어려움) |
| **보상** | 정답 시 숫자가 폭죽처럼 터지는 효과 |
| **교육 요소** | 수 개념, 수세기, 수와 양의 대응 |

## 5. 기술 스택

| 구분 | 기술 |
|------|------|
| **프레임워크** | Next.js 14+ (App Router) |
| **언어** | JavaScript (ES2024) |
| **UI 라이브러리** | React 18+ |
| **스타일링** | Tailwind CSS + Stylus Modules |
| **상태 관리** | Zustand |
| **애니메이션** | Framer Motion + CSS Animations |
| **사운드** | Howler.js (효과음/배경음악) |
| **캔버스** | HTML5 Canvas (색칠놀이) |
| **드래그앤드롭** | @dnd-kit (퍼즐) |

## 7. 핵심 기능 요구사항

### 7.1 필수 기능 (MVP)
- [ ] 홈 화면 - 5종 게임 카드 그리드 표시
- [ ] 게임 1 - 색칠 놀이 (기본 도안 5종, 8색 팔레트)
- [ ] 게임 2 - 퍼즐 맞추기 (2x2 퍼즐 5종)
- [ ] 게임 3 - 음악 놀이 (피아노 자유연주 모드)
- [ ] 게임 4 - 풍선 터트리기 (자유 모드)
- [ ] 게임 5 - 숫자 세기 (1~5 문제 10종)
- [ ] 게임 공통 - 클리어 시 축하 애니메이션
- [ ] 효과음 - 터치/정답/클리어 효과음
- [ ] 반응형 레이아웃 - 모바일 최적화

### 7.2 추가 기능 (v2)
- [ ] 난이도 자동 조절 (아이 수준에 맞춤)
- [ ] 진행도 저장 (로컬 스토리지)
- [ ] 부모 대시보드 (플레이 기록 확인)
- [ ] 추가 도안/퍼즐/문제 확장팩
- [ ] 따라하기 모드 (음악 놀이)
- [ ] 미션 모드 (풍선 터트리기)
- [ ] 다국어 지원 (한국어/영어)
- [ ] 플레이 시간 제한 타이머
